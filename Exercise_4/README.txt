ModNetwork.py contains a class which builds the network with the given rewiring
probability using the provided IzNetwork class. Q1.py and Q2.py both depend on this class.

The plots for questions 1 and 2 are in the plots/ directory under the following
names:
  - connectivity_matrix_p=?.eps for question 1a
  - firings_p=?.eps for questions 1b and 1c
  - multiinformation.eps for question 2
